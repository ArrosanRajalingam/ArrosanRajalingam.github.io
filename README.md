## ‚ú® Features

- **Blazing Fast & Lightweight**: Built entirely with pure HTML, vanilla CSS, and minimal JS. Zero complex build systems, framework dependencies, or bloat.
- **Glassmorphism Design**: Sleek UI featuring frosted glass effects, smooth gradients, and subtle background glow animations.
- **Dynamic GitHub Projects Integration**: Automatically fetches and lists recent public repositories using the GitHub API, complete with live search parsing and language filtering.
- **Automatic Light/Dark Mode**: Defaults to a sophisticated dark theme on load, and elegantly transitions to a bright, crisp light theme upon scrolling down (with manual toggle override available).
- **Responsive Navigation**: Includes a compact "Jump to Section" sticky header for easy reading on all device sizes, and an embedded Table of Contents natively integrated within the hero layout.
- **SEO Optimized**: Standardized meta-tags and descriptions tailored for academic discovery.

## üöÄ Deployment

This project requires no build steps or package managers.

1. Fork or clone this repository.
2. Edit `introduction.html` to insert your own biography, research links, and profile pictures (in the `/Asset/` directory).
3. If using GitHub Pages, push to the `main` branch, ensure Pages is enabled in your repository settings targeting the `main` branch, and the site will automatically deploy!
